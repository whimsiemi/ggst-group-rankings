# GGST - Silli Leaderboards

## DESCRIPTION 
Make an online leaderboard featuring a group of players, from friends and local scene members to national pros and even just your collection of alts!

## SCOPE
This repository is non-functional beyond the command line so far, but it will have the following features when it is finished and available on a website:
- A user interface to create groups of players using an optional display name, their puddle-farm ID and the character they play
- A webpage index for your leaderboard, on which you can refresh the ratings (with a 60-second cooldown) or edit the players in the group (if you created the group on this IP address)
